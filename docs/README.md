---

##  Final Report Sections

The `final_report.md` covers all 8 required capstone sections:

1. **Title and Author** — Project metadata and links
2. **Background** — Problem context, motivation, and research questions
3. **Data** — Dataset descriptions, data dictionary, and variable definitions
4. **EDA** — Exploratory data analysis, visualizations, and data cleaning steps
5. **Model Training** — Algorithms, training strategy, and performance metrics
6. **Application** — Streamlit web app walkthrough with screenshots
7. **Conclusion** — Summary, limitations, lessons learned, and future directions
8. **References** — All cited sources and documentation
