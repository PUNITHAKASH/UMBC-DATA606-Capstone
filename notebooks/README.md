
#  notebooks/

This folder contains all Jupyter Notebooks used for data exploration, model training, and evaluation in the **AgroIntel** capstone project.

---

##  Notebook Index

| Notebook | Purpose | Key Output |
|---|---|---|
| `01_EDA_crop.ipynb` | EDA on Crop_recommendation.csv | Distribution plots, correlation heatmap |
| `02_EDA_irrigation.ipynb` | EDA on irrigation_prediction.csv | Irrigation need breakdown, feature relationships |
| `03_model_crop.ipynb` | Training the crop recommendation model | `crop_recommender.pkl` |
| `04_model_irrigation.ipynb` | Training the irrigation classification model | `irrigation_model.pkl`, `encoders.pkl` |
| `05_model_iot.ipynb` | Training the IoT valve actuation model | `iot_model.pkl` |
| `06_model_evaluation.ipynb` | Comparing all model performance | Accuracy, Precision, Recall, F1-Score |
